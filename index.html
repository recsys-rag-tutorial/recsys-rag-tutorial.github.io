Retrieval-Augmented Generation for Recommender Systems

A tutorial on retrieval, augmentation, and generation for LLM-based recommendation

Overview Motivation Schedule Materials Speakers

Overview

This tutorial introduces Retrieval-Augmented Generation (RAG) for recommender systems. We discuss how retrieval, external knowledge, user history, item metadata, and large language models can be combined to build more accurate, explainable, and interactive recommendation systems.

Target audience: researchers and practitioners interested in recommender systems, information retrieval, and large language models.

Motivation

Large language models have shown strong reasoning and generation abilities, but they often lack personalized, up-to-date, and domain-specific knowledge. RAG provides a practical framework for grounding LLM-based recommendation in retrieved user, item, and contextual information.

  • How can retrieval improve LLM-based recommendation?
  • What should be retrieved: users, items, reviews, knowledge graphs, or demonstrations?
  • How should retrieved information be represented and injected into LLMs?
  • How can we evaluate RAG-based recommender systems?
